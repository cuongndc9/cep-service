## Prepare

- [Node](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/)

## How to use?

```sh
docker run -it -p 5432:5432 -e POSTGRES_USER=postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=postgres postgres
yarn
yarn start
```
